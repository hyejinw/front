정규식(Regular Expression)

문자열 : 

    형식 : / ~~~~~~ / 플러그인
    플러그인 :
      g(global) : 전체 범위에서의 처리  
      m(multiline) : 여러 줄에 걸쳐 처리(textarea 태그에서 사용)
      s(single line) : 한줄만 처리(기본값)
      i : 영문자의 소문자와 대문자의 구별을 하지 않는다
      u(unicode) : 유니코드에 대한 처리
    
      정규식을 실행하기 위해서 사용되는 함수?
        1. 사용자 입력 문자열.match(정규표현식 변수) : 포함 유무에 대한 비교시 편리
        2. 정규표현식 변수.test(사용자 입력 문자열)  : 구체적인 문자(열) 간의 비교시 편리 frequently used
